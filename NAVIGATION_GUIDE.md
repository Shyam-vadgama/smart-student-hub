## ğŸ” Can't Find the Tabs?

### Issue 1: Not Logged in as Principal
**Solution:** Make sure you're logged in with Principal role credentials

### Issue 2: On Mobile/Small Screen
**Solution:** 
- Tabs might be stacked vertically
- Scroll down to see all tabs
- Rotate device to landscape mode

### Issue 3: Page Not Loaded Completely
**Solution:**
- Wait for page to fully load
- Refresh the page (F5)
- Clear browser cache

### Issue 4: Wrong Dashboard
**Solution:**
- Check URL: should be `/principal-dashboard`
- Make sure you're not on Admin/HOD/Faculty dashboard
- Check sidebar title shows "Principal Dashboard"
